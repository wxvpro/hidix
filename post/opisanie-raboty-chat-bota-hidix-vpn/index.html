Мои подписки

В этом меню располагаются конфигурационные файлы подписок, они же ссылки для подключения VPN сервиса. Для активации ключа вам необходимо нажать на ссылку, она скопируется, после же вставить в ваше VPN приложение Инструкции к приложениям:

V2RayTun  Hiddify

Объединить подписки

Ваши активные подписки будут преобразованы в один удобный ключ. Теперь в любой программе вы сможете активировать необходимую вам подписку из всех приобретенных

Помощь

Для современных телефонов мы создали автоматическую настройку. Просто нажмите кнопку и все заработает

Управление ключами

Здесь вы сможете изменить сервер для одного из ваших ключей. Например - вы изначально выбрали сервер одной страны, но хотите попробовать другой

Мои платежи

Здесь вы можете увидеть все свои платежи, совершеннее через платежные системы

Мой баланс

Здесь отображается ваш внутренний баланс, вы можете его пополнить и даже поделиться.

  • Способы пополнения:
  • Оплата через платежную систему
  • Пополнение с помощью кода оплаты, взятого у менеджера
  • Оплата с применением промо кода

Инструкции

В этом разделе предоставлены инструкции по подключению VPN для различных устройств

Пробный период

Для каждого нового пользователя мы дарим пробный период 3 дня на тест любых серверов без ограничений

Тарифы

Здесь располагаются тарифные планы, а в самом низу - список доступных серверов. Выбирайте понравившийся сервер, в последствии (если будет необходимо) его можно будет заменить в личном кабинете «управление ключами»

Выбор сервера

Выберите понравившийся сервер, выберите срок аренды и порядок оплаты

Оплата

Оплатить можно через платежные системы (если они подключены), либо получить код оплаты через менеджера в телеграм

Техподдержка

При переходе в эту область у вас будет возможность написать вопрос техподдержке, поддержка ответит вам в этом же чате

Реферальная программа

Приглашай друзей и получай бонусы на свой баланс. После этот бонус ты можешь использовать для оплаты услуг, или поделиться им с друзьями
